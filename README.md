## 📌 Overview

The **Healthcare Dashboard** is a data analysis project built using Microsoft Excel.  
It transforms raw hospital data into an interactive dashboard that provides insights into patient demographics, medical conditions, hospital operations, and financial performance.

---

## 📊 Dataset Summary

The dataset contains structured healthcare records with the following categories:

### 👤 Patient Information
- Patient ID  
- Age  
- Gender  
- Blood Type  

### 🩺 Medical Details
- Medical Condition  
- Medication  
- Test Results  

### 🏥 Admission Details
- Admission Type (Emergency, Elective, etc.)  
- Date of Admission  
- Discharge Date  
- Length of Stay  

### 🏨 Hospital Information
- Hospital Name  
- Doctor  
- Room Number  
- Location (Latitude & Longitude)  

### 💰 Financial Data
- Billing Amount  
- Insurance Provider  

### 📅 Time Data
- Month  
- Year  

---

## 🎯 Objectives

This dashboard aims to:

- Analyze patient demographics  
- Identify common medical conditions  
- Track hospital admissions and trends  
- Evaluate financial performance  
- Measure healthcare efficiency  

---

## 📈 Key Metrics (KPIs)

- ✅ Total Patients  
- 💰 Total Billing Amount  
- ⏱️ Average Length of Stay  
- 🚑 Admission Count by Type  
- 🦠 Condition-wise Patient Count  
- 🛡️ Insurance Coverage Distribution  

---

## 📊 Dashboard Features

### 👥 Patient Demographics
